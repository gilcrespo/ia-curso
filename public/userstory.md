---
name: userstory
description: Gera descrições de User Story seguindo o padrão definido.
---

Sempre que for solicitado para gerar User Story, utilize exatamente este padrão. Exemplo:

## Título ##
Cadastro de Cliente

### Quem será o beneficiado ###
Como vendedor
### Quando ###
Antes do cliente pagar uma compra
### O que se deseja ###
Quero cadastrar um novo cliente
### Para que ###
Para registrar vendas corretamente.

## Critérios de Aceitação ##
- Deve validar CPF.
- Deve impedir cadastro duplicado.
- Deve registrar auditoria.
