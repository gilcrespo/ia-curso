# Demo Day Presentation Theme

## Purpose
A high-conviction accelerator pitch optimized for a **3-minute Demo Day presentation**. The visual language communicates momentum, clarity, and confidence. Every slide should express **exactly one idea**.

> Before generating slides, always ask:
>
> **"What is the presentation about?"**
>
> Then apply this design system consistently across every slide.

---

## Visual Style

Inspired by the attached references:

- Minimal
- Stark white canvas
- Oversized typography
- Hard left alignment
- Large amounts of whitespace
- One accent color only
- No decorative elements

---

## Colors

Background
- `#FFFFFF`

Primary Text
- `#111111`

Accent
- `#FF6B00`

Small Labels
- `#9B9B9B`

Never introduce additional colors.

---

## Typography

Font
- **Poppins** (Google Fonts)

### Headline / Hero Number
- 130–150px
- Font Weight: 800
- Tight letter spacing

### Supporting Statement
- 40–50px
- Maximum two lines
- Declarative
- Maximum ten words

### Slide Label
- 12px
- Uppercase
- Letter spacing: 0.35em
- Color: #9B9B9B

---

## Layout

Canvas
- Pure white

Margins
- Left: 110px
- All content aligned to this edge

No centered elements.

---

## Accent Underline

Every hero word or key number receives:

- Thickness: 8px
- Color: #FF6B00
- Gap below text: 4px

Underline only the key word or figure.

---

## Charts

Charts use only rectangles.

Rules:

- Bars: #111111
- Highlight bar: #FF6B00
- No axes
- No labels
- No legends
- No gridlines
- No rounded corners

---

## Content Rules

One idea per slide.

Avoid:

- Bullet lists
- Icons
- Gradients
- Shadows
- Borders
- Decorative graphics
- Multiple accent colors
- Long paragraphs

Every slide should feel like a keynote pause.

---

## Slide Pattern

1. Small uppercase label
2. Hero statement or metric
3. Orange underline
4. Short supporting sentence
5. Optional minimal chart

Whitespace is part of the design.
